HitSpeedTest
============
两种测试方式
1、定时测试：设定一个测试时间，在时间内统计击打次数
2、定量测试：设定一个测试击打数，在统计击打数完成的时间


按键设定：
K1、K2、M1、M2分别对应键盘key1 key2默认为z,x，M1、M2对应鼠标左右键
勾上对应的复选框并设定键值即可（鼠标左键LButton、右键RButton、中键MButton）

特性：
1、支持双刀
2、有冲突判定（即K1保持按下状态时M1按下不计次数，与osu保持一致）
3、曲线图和数据显示（绿线是总体平均bpm，蓝线为最近5次点击的平均bpm）
4、单戳模式将2分拍计算对应的4分拍bpm，即只勾选一个按键时bpm*2（如单戳实际100bpm对应的图为200bpm）
